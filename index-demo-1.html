Full Stack Social Media App

Unsplash
Api

I built an image browsing application using the Unsplash API where images first load in a blurred state and then transition smoothly to full resolution for a better user experience. The search functionality is optimized using debouncing to reduce unnecessary API calls, and I implemented pagination for efficient data handling. The application state is managed using the Context API,.

FeedFusion

FeedFusion is a full-stack MERN app with role-based access (User, Admin, Super Admin) secured by access token and refresh token authentication. It features personalized feeds, Admin content management, post saving/liking, frontend caching, lazy loading, pagination, and Redux Toolkit with persisted state. Built for speed, security, and scalability.
